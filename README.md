# Bulls and Cows Game

A number guessing game for Telegram and Web.

## Play Online

🎮 **Play the game:** https://raczek-dev.github.io/test-repo

## How to Play

Guess the 4-digit secret number!
- 🎯 **Bull** = correct digit in correct position
- 🐄 **Cow** = correct digit in wrong position
